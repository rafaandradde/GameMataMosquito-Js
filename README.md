# GameMataMosquito-Js

## O que é este projeto?

Este é um jogo que desenvolvi como parte de um curso de JavaScript. É um Game simples mas que utiliza muita lógica em Js.


## Funcionamento do Jogo

Este é um jogo de exterminador de moscas, onde o jogador assume o papel de um exterminador armado com uma mosqueteira. O objetivo do jogo é esmagar as moscas que aparecem na tela dentro de um limite de tempo definido.

![image](https://github.com/rafaandradde/GameMataMosquito-Js/assets/147434748/b3d995bc-7ea5-478a-9681-2f0d876110d2)


1. **Escolha do Grau de Dificuldade**: O jogador começa o jogo com a opção de escolher o grau de dificuldade. Isso determina o número de moscas que aparecerão na tela e o tempo disponível para esmagá-las.

2. **Esmagando as Moscas**: As moscas surgem aleatoriamente na tela, e o jogador deve clicar sobre elas usando a mosqueteira para esmagá-las.

3. **Vitória**: Se o jogador conseguir esmagar todas as moscas dentro do tempo estabelecido, ele vence o jogo.

4. **Game Over**: No entanto, se o jogador permitir que 3 moscas escapem sem serem esmagadas, o jogo termina com a mensagem "Game Over".

5. **Níveis de Dificuldade**: Quanto maior o nível de dificuldade escolhido, mais moscas aparecerão na tela em um intervalo de tempo menor, tornando o desafio mais emocionante.

Divirta-se e desafie suas habilidades tentando esmagar todas as moscas dentro do limite de tempo para alcançar a vitória!
